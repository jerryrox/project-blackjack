"_project_" directory contains the source files of all assets used in the game.

All assets used in development of this game were made by myself, or by third parties which granted permission to use them in my game.

If you'd like to test the game with unlimited gold, you may either:
1. Modify the database table (username = jerryrox, password = jerryrox)
2. Enter with runtime mode 'cli' and use 'setGold' command to modify your balance. Restart the game to relaunch in gui mode.
